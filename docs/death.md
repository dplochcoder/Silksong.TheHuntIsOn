# Death Module

Configure penalties for any Hornet who dies.

## Respawn Timer

A delay, in seconds, the deceased player must wait before respawning. Respawn timer does not advance while the server is paused.

## Spawn Coccoon

If turned off, players do not spawn a coccoon on death nor suffer any associated penalties (loss of rosaries, limiting silk). This overrides the following settings.

## Lose Rosaries

If turned off, players do not lose any rosaries on death.

## Limit Silk

If turned off, players do not have their silk limit reduced back to 9 on death.
