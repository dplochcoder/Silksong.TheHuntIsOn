# Modules

Modules control various aspects of the game, and can be configured for a specific team, for both teams, or for each team individually.

Settings can be tested instantly when playing locally, but when playing on an SSMP server, they must be applied to the server via the 'Send Settings' option. The server will then broadcast the new settings to all clients, updating them. Only authorized users are allowed to change module settings for the server.
