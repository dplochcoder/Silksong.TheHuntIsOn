# Bind Module

The bind module alters balance parameters around the Bind ability, the primary form of healing.

## Heal Masks

The number of masks a successful Bind action heals.

## Multibinder Heal Masks

The number of masks a successful Bind with the Multibinder equipment heals.

## Silk Cost

The number of silk spools required to perform a Bind action.

Note that lowering this to 3 or below still will not allow Cursed players to Bind.

## Time Penalty

A multiplier on the length of time it takes to successfully exeute a Bind.  (Default 1.2s)
