# Silksong.TheHuntIsOn

An [SSMP](https://github.com/Extremelyd1/SSMP) add-on for "Speedrunner vs. Hunters" in [Silksong](https://store.steampowered.com/app/1030300/Hollow_Knight_Silksong/).

Functionality is broken down into different [modules](https://github.com/dplochcoder/Silksong.TheHuntIsOn/blob/main/docs/MODULES.md), which can each be enabled or disabled independently of one another.
